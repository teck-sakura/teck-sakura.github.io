Q リアルミーグリのとき、京都でお土産買った？

→ ちりめん山椒を買いました〜！

Q 大阪行ったことある？

→ １回だけあります！
明日から大阪…たのしみ！


Q 食べたい大阪グルメは？

→ まず、たこ焼き。そして大阪グルメって何があるかなって調べて見つけた「ねぎ焼き」‪🌿


Q 最近みた映画ある？

→ 「ある男」です、面白かった！
映画の趣味はももちゃんと合うことおおい〜


Q とり吉は何歳？

→ 3歳、人間でいう33歳です
同い年の方いますか？


Q これまで呼ばれたことのあるあだ名

→ 呼ばれて嬉しかったのは「なかがわんわん」
でも長くて定着しなかった🐶


Q ひつじのショーン誰が1番すき？

→ ショーンが1番好き。 その次にティミーのママと映画に出てくるルーラです！


Q 長崎の好きなところは？

→ 夜景がきれいなところ🏙
私がVlog撮影で行った稲佐山の展望台、夜に行くとキラキラしてて綺麗なんだよ〜


Q 運動と学力どっちが自信ありますか？

→ うーーん、正直どっちも自信ナシ…
でも、頑張りたい気持ちはあるんです！

特に運動能力チェックで陸上選手の山田桃実を抑えて1位を取ったら絶対かっこいいよね？
今から鍛えようかな？


Q 夏らしいこと何かしたー？

→ 璃花さんとかき氷を食べに行きました🍧
しあわせな時間でした…！




Q かき氷の好きな味は？

→ レモン！ でも果物がたくさん乗ってるかき氷はぜーんぶすきかな


Q MBTIは？

→ さっきやってみたらISFPでした🧚‍♂️
自由を愛する性格らしいです、多分合ってる


Q ひとりカラオケに行った時歌う曲は〜？

→ 松田聖子さんの赤いスイートピーはほぼ毎回歌っている気がします！あとコレサワさんの曲をよく歌うかな〜


Q 辛い食べ物は得意？

→ おでんにからしをたくさん付けて食べるの大好き！でもなんていうのかな、ホットな辛さは得意じゃないかもしれない…！


Q 東京での生活慣れた？

→ はい！！ でもまだ稀に道に迷うことがあるから、四期生のみんなを頼ってます🤝🏻
