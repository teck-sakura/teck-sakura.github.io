佐賀県出身 18歳 佐藤愛桜です🌸



はるさんからバトン受け取りました~



FUJIYAMAで大好きだよーーと告白された時はすごく驚きました！

カメラマンさんを発見していたはるさん📸
私は一度も目を開けることが出来なかったので本当に尊敬でしかないです！

頼もしいはるさんとペアで、すごく心強かったよ〜




9月に入り、涼しいなぁ。もう秋だなぁ。
と思っていましたが、昨日セミの鳴き声を耳にしてしまいました~





もう少し夏は、続きそうですね🎐꙳⋆ 



お洋服も早く秋物を着たいのに、まだ暑いし
なんか難しいですよね( ; ; )
この前、お洋服を買ったんですけど、今の暑い季節も着れて、これから先も着れるようなセットワンピースを買いました！
可愛くてお気に入りなんです♡♡






まだまだ暑いので熱中症には気を付けて下さいね☺︎


⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅




9月14日に放送された
「そこ曲がったら、櫻坂？」は見て頂けましたか~?




「四期生リアクションチェック！後半」
では、戦慄迷宮に行ってきました🧟‍♀️




人生初の戦慄迷宮!!




実は、、お化け屋敷も初めてでした👻




お化け屋敷なんて怖くない!!と思っていましたが、いざ建物を前にするとすごく不気味で、寒気も感じてしまいドキドキハラハラでした😖




1番怖かったのは、廊下です、、




横から来るだろうな、、と思っていたので
後ろから現れるとは思っておらず( ; ; )




すごく驚いてしまいました
(なかなか伝わらなかったですよね…)




戦慄迷宮では、「怖い無理かも( ; ; )」「意外と大丈夫かも!!」の繰り返しでした…
でも、ペアのういとずっと一緒にいたので意外と行けてしまい、あっという間にゴールしていました




もっと怖い所をお伝えできたら良かったのに、淡々と終わってしまったので大反省です😭😭




リアクションチェックを通して、私はリアクションが薄いなと思ってしまいましたが、たくさんのことを学んだので、これからもっと頑張っていきます(ง •̀_•́)ง




みなさん、「ちょこさく」も見て頂けましたか？
こちらにも出演させて頂き、人生初の食レポをしたんですが、思っていたより難しくて




三期生の先輩方がフォローをして下さり、面白くコメントして下さったので、先輩の偉大さを改めて感じました☺︎




色んな意味でまだまだ勉強が足りないので、
バラエティーをマスターしていけるようになりたいです！





⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅ ⑅




ここからは、みなさんから頂いた質問に答えていきたいと思います☺︎




✿朝は強い？弱い？

▶強い方だと思います！
朝が早い時は、「起きれるかな…??」と不安な気持ちの方が勝ちアラームより早く起きてしまいます☺︎

しっかり者のももちゃんにも起きてなかったら
起こして欲しいと頼まれることがあります^^
(頼られてなんだか嬉しいです♪)




✿好きなおにぎりの具材は？

▶1つに絞れないほど沢山あります🍙
特に好きなのは…鮭、焼きタラコ、ツナ、昆布、梅、エビマヨです!!
私はお米が好きなので、塩おにぎりを選ぶことも多いです~




✿最近驚いたことは？

▶1日に7km歩いたことです！
1人で知らない道をお散歩していたのですが、 気づいたら迷子になってしまい、、
知らない道を黙々と歩いているとすごい距離を歩いていました😳
知らない道をナビを見ずに歩くことは、少し怖
いですが、新たな発見が沢山あるので楽しいで
す♡
